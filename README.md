# Docker Box (개인 Docker 프로젝트 설정 저장소)

이 저장소는 **개인이 자주 사용하는 Docker 컨테이너들을 효율적으로 관리**하기 위해 구성된 맞춤형 프로젝트 설정 모음입니다.

이는 사용자 본인의 개발 환경에 맞게 커스터마이징된 스크립트와 환경 변수 설정을 통해, 여러 개의 개별 컨테이너를 일관된 방식으로 빠르고 편리하게 구동, 중지, 관리할 목적으로 사용됩니다.

## 💡 프로젝트의 핵심 목표

1.  **일관성 있는 환경 관리:** 데이터베이스, 웹 서버 등 다양한 서비스 컨테이너를 일관된 명령 체계로 관리합니다.
2.  **환경 변수 중앙 집중화:** 서비스별 비밀번호, 사용자 정보, 포트 설정 등을 하나의 설정 파일 템플릿으로 통합하여 관리합니다.
3.  **간편한 컨테이너 생명주기 제어:** 스크립트를 통해 컨테이너의 시작(Start), 중지(Stop), 제거(Remove)를 단순화합니다.

## 📦 포함된 서비스 예시

* 데이터베이스 (MariaDB, MongoDB, PostgreSQL 등)
* 캐시 및 메시징 큐 (Redis, RabbitMQ 등)
* 웹 서버 (Nginx, Apache 등)
* 범용 OS 환경 (Ubuntu, CentOS 등)
